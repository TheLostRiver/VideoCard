# Project State

更新时间：2026-05-01

## 当前目标

构建一个静态 HTML 版交互式游戏显卡天梯图。首版覆盖 NVIDIA GTX 10 到 RTX 50、AMD RX 400 到 RX 9000、Intel Arc A/B，包含桌面版和移动版显卡。移动版必须独立标注。当前新增目标是本地后台编辑器优先的数据维护能力，暂不引入 PostgreSQL。
